# Repositorio do artigo de realidade aumentada sobre o OTTO para ERI 2023 

Este código faz parte do Projeto de Iniciação Científica (PIBIC) da Universidade Federal de Mato Grosso (UFMT), é um projeto de Realidade Aumentada Interativa que permite exibir modelo 3D interativo do ROBO OTTO em um ambiente real através da câmera do dispositivo apontada para a imagem do otto.
